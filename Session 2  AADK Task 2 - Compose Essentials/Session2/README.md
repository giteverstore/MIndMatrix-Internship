# Session 2 - AADK Task 2: Compose Essentials

## Overview
This project demonstrates Jetpack Compose fundamentals including:
- Layout Primitives (Box, Column, Row)
- Modifier chaining
- State management
- Responsive UI testing
- Accessibility enhancements
- Dark mode support

## Features Implemented
- Profile Card UI
- Follow button with dynamic state
- Material3 Card with elevation
- Responsive preview (multiple screen sizes)
- Dark mode preview
- Accessibility using contentDescription

## Layout Primitives Used
- Box → For screen centering
- Column → Vertical arrangement
- Row → Horizontal arrangement
- Spacer → Controlled spacing

## Modifier Usage
- fillMaxSize()
- fillMaxWidth()
- padding()
- weight()
- semantics()

## Accessibility
- contentDescription added to interactive elements
- Dark mode compatibility tested
- Typography hierarchy maintained
